Guide:nuc8i5beh install black apple tutorial, close to perfect operation

发表于2019-05-16|更新于2020-07-09|黑苹果指南
|阅读量:|评论数:

Purchase hardware facilities

  • nuc8i5beh

  • Magnesium m.2 1100 256g solid state hard drive

  • Samsung DDR4 2400 notebook memory

  • HKC computer monitor

  • 8G U disk (this was previously, not part of the purchase)

所需软件

  • Mojave 10.14.2
    Baidu network disk

  • balenaEtcher：
    Official website

  • bios65 file：
    download link

  • nuc8 Installation package：
    network disk Extraction code：i21t

  • EFI file

    Github address

安装过程

  1. Insert a USB flash drive under Windows，Start balenaEtcher，“select image”select“Mojave 10.14.2”Mirror,Select U disk，Click on “Flash”

note：

  • BIOS update to version 0056
  • Select APFS partition on the SSD
  • It is best to install 10.14.2 first, then upgrade
  • The mac version can be upgraded to the latest version 10.14.4 (the blogger has been updated to mac mojave beta 10.14.5)

BIOS settings

  1. Copy [BE0056.bio] in the downloaded BIOS56 file to a blank U disk root directory.
  2. Start nuc, press [F7], select BE0056.bio, and update the BIOS.
  3. After the update, start nuc again, press [F2] to enter the BIOS setup interface.
  4. The BIOS settings are as follows:
    • « Inter VT for directed I/VO (VT-d) » ： disabled
    • « Secure Boot » ： disabled
    • « Legacy Boot » ：enabled
    • « Fast Boot » ： disabled
    • Boot->Boot Devices-> « USB » ： enabled
    • SATA mode ： AHCI
    • Boot->Boot Configuration-> « Boot Network Devices Last » ： disabled
    • Power->Secondary Power Settings, « Wake on LAN from S4/S5 », set to « Stay Off »
    • Devices->Video, « IGD Minimum Memory » set to 64mb
    • Devices->Video, « IGD Aperture Size » set to 256mb

note：

Do not tick for disable, tick to enable

Boot the macOS installer on NUC using Clover UEFI

  1. Insert the USB flash drive into the computer where the system needs to be installed.
  2. To display the Clover boot interface, you need to set the FakeID (Clover Options -> Graphics Injector -> FakeID)
    • Click the « Enter » button to modify it and change it to： 0x12345678
    • Press « Return » twice to return to the Clover boot interface
  3. Select « Boot macOS Install from install_osx » and press « Enter »
  4. The installation interface appears. Select [Disk Tools] to format your SSD as [APFS] format.
  5. Select “install mac OS”, install the system. [restart, please perform step 2 again]
  6. After the installation is successful, enter the system
  7. Open Clover Configurator and select Mount EFI in the list on the left, as shown below:

  1. Click [Mount Partition] under [Efi Partitions], enter the password, and click [Open Partition] to open the EFI folder.
  2. Copy the [BOOT] folder and the [CLOVER] folder in the USB flash drive to the EFI folder of [Step 8]. (This way you no longer need to plug in the U disk when you start the system)
  3. Under this link, download [config_nuc8_bc.plist]:https://github.com/RehabMan/Intel-NUC-DSDT-Patch
  4. After downloading, rename it to: [config.plist] and copy it to the [CLOVER] file in the EFI folder of [Step 8]. Clip it and replace it.
  5. If you don’t understand, you can refer to the tutorial of foreign gods.：Great God Tutorial

  2. Thunder 3 does not support hot swap
